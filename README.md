# Mini-Projet DevOps - Formation App

## Description
Cette application web a été développée avec Flask et SQLite.
Elle permet d’afficher et d’ajouter des formations via une interface web simple.

## Fonctionnalités
- Page d’accueil
- Liste des formations
- Ajout d’une nouvelle formation
- Base de données SQLite

## Technologies utilisées
- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- Docker
- Docker Compose
- GitHub Actions
